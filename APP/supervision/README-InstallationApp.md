#Installation de l'application de supervision.
Pré-requis : avoir réalisé l'installation du serveur.
1. Cloner le projet depuis le répertoire github : https://gitlab.univ-lr.fr/stage2021_aladenise_elie/seriousgame
2. Se placer dans la branche `test-app-web` puis aller dans le répertoire APP à l'aide d'un terminal
3. Installer les dépendances avec `yarn install` ou `npm install`
4. Lancer l'application avec `yarn run start` ou `npm run start`
5. L'application devrait s'ouvrir alors automatiquement sur votre navigateur internet autrement, elle est disponible ici : `http://localhost:3000/`
